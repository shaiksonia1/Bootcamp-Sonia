Project Notes for Exercise 2: CSS Variables and Basic Styling

Overview of Exercise 2

This exercise focuses on learning how to use CSS variables to manage styles efficiently. The project involved creating a simple webpage and applying CSS variables to handle colors, fonts, and spacing. Below are the details of the steps taken to complete this exercise, along with the challenges I faced during development.

Thought Process

Here's how I approached each part of the exercise:

  • Solution 1: In this step, I focused on creating a simple webpage with a clean structure. I defined variables for primary and secondary colors, font sizes, and margins in a separate CSS file. This made it easier to maintain consistency across the page and quickly apply style changes.
  • Solution 2: For this part, I added more styling by using default values for some variables and styling a table with alternating row colors using `nth-child`. I also learned how to use `min()`, `max()`, and `clamp()` to create responsive font sizes.
  • Solution 3: This step introduced more advanced concepts, such as defining variables based on other variables (e.g., `--secondary-color: var(--primary-color)`). I also implemented a theme switcher to toggle between light and dark modes, which made the webpage more dynamic and user-friendly.

Challenges Faced

During this exercise, I faced the following challenges:

  • Understanding how to use CSS variables for managing colors and layout efficiently was initially tricky.
  • Finding the right balance in font sizes and spacing to ensure readability across devices was a challenge, especially when using responsive units like `vw` and `vh`.
  • Implementing the theme switcher required understanding how to link multiple CSS files and toggle between styles without complex JavaScript.
