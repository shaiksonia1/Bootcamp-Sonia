Thought Process

Here's how I approached each solution while building this project:

  • Solution 1 - The first step was creating a basic webpage with an ordered list of my favorite movies and an unordered list of books. I added an image to make it more visually appealing. The focus here was to build a clean and simple structure, ensuring that the content was easy to follow and read.
  • Solution 2 - In this step, I introduced more complexity by adding two side navigation bars. These navigation bars allowed for more options and links, which made the page look more dynamic. I used inline and internal CSS to position and style the elements.
  • Solution 3 - This version expanded on the previous one by adding detailed content. I included descriptions for movies and books and displayed a table for additional information. The main challenge here was keeping everything well-organized while maintaining readability, especially with the added navigation bars. Challenges Faced

During the development of this project, I faced a few challenges:

  • Positioning the left and right navigation bars correctly was tricky without using CSS frameworks. I had to experiment with different float settings to get them right.
  • Making sure the page content was spaced well and easy to read, especially when I added more sections, was a bit challenging.
  • Adding more details to the content while keeping the page clean and simple, particularly in the movie and book lists, was a bit tricky.
