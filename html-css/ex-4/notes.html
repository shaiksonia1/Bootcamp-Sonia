Project Notes for Exercise 4: CSS Units and Responsive Design

Overview of the Exercise 4

This exercise focused on using various CSS units and responsive design techniques to create flexible and scalable web pages. The goal was to work with different units like px, em, rem, vh, vw, and functions like min(), max(), and clamp() to create fluid and responsive layouts.

Thought Process

Here's how I approached each level in Exercise 4:

  • Level 1 - In the first level, I created a basic centered webpage layout using percentage-based widths and viewport units like vh and vw to center the content horizontally and vertically. The goal was to create a flexible layout that adapts to various screen sizes.
  • Level 2 - For the second level, I explored the use of min(), max(), and clamp() functions for font sizes and widths. These functions allow for scalable designs where content adjusts according to the viewport size, making the page more readable on different screen sizes.
  • Level 3 - In the third level, I implemented a flexible grid system and fluid typography to make the layout and text responsive without relying on fixed breakpoints. This involved using CSS Grid and clamp() for font sizes to ensure content scales appropriately on all devices.

Challenges Faced

During the development of this project, I encountered a few challenges:

  • Understanding how to effectively use viewport units like vh and vw took some time, as these units scale based on the size of the viewport and not the container.
  • Implementing fluid typography with clamp() was a bit tricky at first, but it proved to be a powerful way to make text responsive without breaking the layout.
  • Creating a fully responsive grid system without relying on fixed breakpoints required experimenting with CSS Grid properties like minmax() and auto-fit.

Key Learnings

This exercise helped me learn how to use CSS units and responsive design techniques to create flexible and scalable layouts. I learned how to use viewport units, the clamp() function for fluid typography, and how to design using flexible grids without fixed breakpoints. These techniques help ensure that web pages are adaptable to various screen sizes, from mobile devices to large desktop monitors.
