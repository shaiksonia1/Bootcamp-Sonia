Exercise 3: Logging with Winston

Level 1

Problem Description:

  • Install winston and create a logger with info, warn, and error levels.
  • Log events like "Program started" and "Operation successful".
  • Division by zero should log as a fatal error.

Exit Criteria: Logs appear in the console with appropriate levels.

Level 2

Problem Description:

  • Add file transport to save logs to a file.
  • Include timestamps in the log format for readability.

Exit Criteria: Logs are saved to a file with clear timestamps.

Challenges I Faced

  • Configuring multiple transports and log levels in winston.
  • Formatting log outputs with timestamps.

Improvements

  • Documenting logging setup for future reference.
  • Adding custom log levels for better categorization.
