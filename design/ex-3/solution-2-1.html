Exercise 3: Basics of Icons (Level-1)

Home page

To add and style icons using Font Awesome, I included the Font Awesome library in the head of the HTML file. Then, I added three icons to the page using i tags with specific classes, like `fa-brands fa-youtube for a youtube icon. I used CSS to change the size, color, and spacing of the icons to make them stand out. This helped me learn how to use and customize icons to improve the design of a webpage.
