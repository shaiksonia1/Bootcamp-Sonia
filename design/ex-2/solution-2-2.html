What I have implemented

I created a webpage that allows users to toggle between a dark and light theme. In the HTML, I included meta tags for the page's character set, viewport settings, and description. I added links to the CSS files for both themes and used a JavaScript function to switch between them when the user clicks the toggle button.

In the CSS, I defined two themes—light and dark—using CSS variables for colors like primary, secondary, background, and text. The light theme uses lighter background and darker text, while the dark theme uses darker background and lighter text. I styled the overlay text on the image to be centered and readable on both themes using absolute positioning and transform.

I also demonstrated HSL color variations by adjusting saturation and lightness, creating a range of colors. To enhance the typography, I used custom fonts with @font-face to allow medium and bold weights. For layout, I used flexbox to create a responsive card design with padding, rounded corners, and subtle shadows. The buttons have transition effects on hover to fit the theme style.
